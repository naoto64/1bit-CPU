# 1bit CPU 組み立てキット

## 概要
4個のロジックICを使用した1bit CPUの組み立てキットです。

## スペック
|項目|スペック|
|---|---|
|汎用レジスタ|1bit x 1|
|アドレス空間|1bit|
|ROM容量|4bit|
|命令セット|ADD, JMP|
|プログラムカウンタ|1bit|
|フラグレジスタ|未実装|
|算術演算|1bitの加算（XOR）|
|クロック周波数|約1Hz|
|IC総数|4個|

## 命令セット
|命令|機械語|説明|
|---|---|---|
|ADD A, Im|0|AレジスタにIm（イミディエイトデータ）を加算する。|
|JMP Im|1|Imで指定した先の番地へジャンプする。|

## プログラム例
### Aレジスタに1を加算し続けるプログラム
```text:
ADD A, 1
JMP 0
```

### Aレジスタに1を加算するだけのプログラム
```text:
ADD A, 1
JMP 1
```
### 何もしないプログラム
```text:
JMP 0
```
